Productos para Artes Gráficas

Fabricamos productos de alta especialidad para la industria de las artes gráficas, tales como Limpiadores de Rodillos Anilox, Removedores de Tintas para Placas de Fotopolímero, Limpiadores de Mantillas, etc., con bajo contenido de solventes y no inflamables.

Limpiadores para Flexografía

Productos especializados para la limpieza de equipos de flexografía:

CLEANILOX

Limpiador especializado para rodillos anilox. Remueve eficientemente residuos de tinta seca y acumulada en las celdas de los rodillos, restaurando su capacidad de transferencia de tinta.

FLEXOMATE

Removedor de tintas para placas de fotopolímero. Formulado con bajo contenido de solventes, es una alternativa segura y eficiente para la limpieza de placas flexográficas.

Limpiadores para Offset

ECOSOLVE-E7

Limpiador de mantillas y rodillos para prensas offset. Formulación de alta eficiencia con bajo contenido de solventes y punto de inflamación elevado para mayor seguridad en el área de trabajo.

PLAX-400

Limpiador especializado para placas de impresión offset. Remueve residuos de tinta, goma y otros contaminantes sin dañar la imagen de la placa.

Productos Generales

ECOSOLVE-E5

Solvente dieléctrico multipropósito para limpieza general en la industria de artes gráficas. No inflamable y seguro para el operador.

ECOCREAM-INK

Crema limpiadora para la remoción de tintas de las manos. Fórmula suave que no irrita la piel, ideal para el uso diario en talleres de impresión.
